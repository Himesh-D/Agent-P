# 🛡️ AGENT - P | Advanced Complaint Management System

![AGENT-P Banner](favicon.png)

AGENT-P is a high-performance, minimalist, and secure complaint management system designed to streamline communication between users and administrative teams. Built with modern web technologies, it features a stunning glassmorphism interface and real-time tracking.

## ✨ Key Features

- 💎 **Glassmorphism UI**: High-end visual design with real-time blur and sophisticated gradients.
- 🚀 **Interactive Particles**: A dynamic and professional background that responds to user interaction.
- 📊 **Admin Dashboard**: Real-time stats visualization (Total, Pending, Resolved) with immediate ticket updates.
- 🆔 **Ticket System**: Automated ticket ID generation for easy tracking.
- 📄 **PDF Export**: Generate professional resolution reports with a single click using jsPDF.
- 📱 **Mobile Responsive**: Fully optimized for desktop, tablets, and mobile browsers.
- 🛡️ **SEO Optimized**: Built-in best practices for visibility and structure.

## 🛠️ Built With

- **HTML5 & CSS3**: Custom modern design with Flexbox and Grid.
- **JavaScript (Vanilla)**: High-performance logic and state management.
- **Lucide Icons**: Consistent, high-quality iconography.
- **Particles.js**: For the immersive background experience.
- **jsPDF**: For generating professional reports on-the-fly.

## 🚀 Getting Started

To get the system running locally:

1. **Clone the project** or download the source code.
2. Ensure you have the `favicon.png` in the root directory.
3. **Open `index.html`** in any modern web browser.
4. No server required—AGENT-P runs locally using browser `localStorage`!

## 📖 Usage Guide

### For Users
- Go to the **User** section from the navigation.
- Fill in your name and the details of your issue.
- Use the **Track Status** search bar to see updates on your past tickets by entering your name.

### For Administrators
- Access the **Admin** dashboard to see all incoming requests.
- Mark tickets as **Resolved** with a click.
- Click **Export PDF** to generate a full report of all records for documentation.

## 📝 Configuration

- Data is stored in your browser's `localStorage` under the key `agent_p_data`.
- This ensures that even after refreshing or closing the browser, your records remain intact.
